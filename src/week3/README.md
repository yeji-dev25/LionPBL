# 과제명
### 객체지향 II - 상속/다형성/추상화

## ⚙️ 실행 방법
- Main.java 파일에서 실행 버튼을 클릭하세요.

## 💡 작업 내용
- 아기사자, 운영진을 입력 받는다.
- 아기사자, 운영진 정보와 함께 과제 제출 여부가 출력된다.
- 아기사자일경우 true, 운영진일경우 false가 출력된다.
- 클래스와의 관계만을 통해 구현하였다.
